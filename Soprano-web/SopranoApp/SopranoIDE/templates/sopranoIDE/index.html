¿Qué es Soprano?

Soprano es un lenguaje de programación musical orientado a la composición algorítmica. Es decir, que compone música mediante algoritmos, usando estructuras definidas y una sintaxis específica. Soprano permite escribir programas sencillos de forma similar a los lenguajes de programación habituales como C, Python o Java. Su gramática está escrita con ANTLR y python.

¿Cómo usarlo?

  1. Elige un nombre corto para tu proyecto.
  2. Elige el nombre de tu procedimiento principal, por defecto es Main.
  3. Presiona F9 para compilar tu programa y generar una canción.

Reglas del lenguaje Soprano

  • El nombre de los procedimientos tiene que empezar con una letra mayúscula, seguida de una o más letras minúsculas o números.
